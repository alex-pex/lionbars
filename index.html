LionBars

A free to use, lightweight jQuery plugin that makes scrollbars look like in OSX Lion - based on Nikolay Dyankov original work.

How to use

  • include css et js files
  • simply use lionbars method on the div(s) you want

Example :

$(function() {
  $('.overflowed').lionbars();
});
  • that's it !
